# Newton-Kepler Protocol (NKP)

A classical four-field substrate simulation that
produces emergent GR-like geometry in the
weak-field limit from self-organizing field dynamics.

## What this is

NKP is a simulation and theory framework built
around four coupled scalar fields (σ, α, ν, J)
that evolve from random initial conditions and
self-organize into an ordered phase.

The ordered phase produces:
- A Yukawa gravitational well from the α field
- An acoustic metric with Schwarzschild-like structure
- Reproduction of three classical GR tests:
  - Light deflection:        1.750 arcsec ✓
  - Perihelion precession:   43.0 arcsec/century ✓
  - Gravitational redshift:  z = Φ/c² ✓

## What this is not

- A derivation of GR from first principles
- A quantum theory of gravity
- A replacement for General Relativity
- Peer reviewed or independently verified

## Methodology

This project was developed by an independent
researcher without formal physics credentials,
working entirely from an iPhone using a
Coherent Multi-Model Collaboration approach —
Claude, Copilot, Grok, and Gemini as thinking
partners, with the author as integration point
and final arbiter of all results.

Every confirmed result is documented with
specific numbers. Every open question is
named precisely. Hypotheses are marked as
hypotheses.
